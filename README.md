# Projeyi Yayına Alma

## Geliştirme

```bash
npm install
npm run dev
```

## Build

```bash
npm run generate
```

## Git İşlemleri

```bash
git add .
git commit -m "Mesaj"
git push origin main
```

## Github Pages Deploy

```bash
npm run deploy
```

## Tam Yayın Alma Süreci

```bash
git add .
git commit -m "..."
git push origin main
npm run generate
npm run deploy
```
